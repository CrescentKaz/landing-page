Things I've Learned:

-> The CSS file must be attached in the Head, not the Header.
-> The JS file must be attached at the bottom of the Main, NOT in the Head.
-> JS doesn't like Python syntax.
-> It's possible to build a website using a separate file per language and also possible to build one with all the CSS and JS embedded in the HTML file.
-> JS can automatically change your HTML based off a variable of your choice (date, random number, time, etc.)
-> Dropshadows increase the visability of text when you have a busy background image.
-> Free artisitic sites like Canva and Google AI Art are your best friend. Just beware the TOS if you plan to monitize anything.
-> The main difference between a static site and a dynamic one is if user-entered data is stored.
-> The correct notation style is the one your current teammates understand.
-> Waterfall is everyone working on the same task at the same time. Agile is divide and conquer.
-> APIs are giant JS dictionaires.
